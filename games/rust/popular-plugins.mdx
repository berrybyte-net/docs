# Best Rust Server Plugins - Essential uMod Plugins

Transform your Rust server with these essential plugins that enhance gameplay, improve administration, and create engaging player experiences. This curated list covers the most popular and reliable plugins used by successful Rust servers worldwide.

All plugins require [uMod framework](/games/rust/install-oxide) installed on your [Rust server](https://berrybyte.net/rust-server-hosting). Each plugin includes installation instructions, configuration tips, and usage examples.

## Admin & Moderation Tools

### Admin Hammer
**Essential building and admin tool**

- **Purpose:** Enhanced building tools for admins
- **Features:** Instant building, no resource costs, advanced placement
- **Commands:** `/ah` - Toggle admin hammer mode
- **Download:** [umod.org/plugins/admin-hammer](https://umod.org/plugins/admin-hammer)

**Quick Setup:**
1. Upload `AdminHammer.cs` to `oxide/plugins/`
2. Grant permission: `oxide.grant user STEAMID64 adminhammer.use`
3. Use `/ah` in-game to toggle

### Vanish
**Invisible admin mode for monitoring**

- **Purpose:** Make admins invisible to players
- **Features:** Complete invisibility, no collision, silent movement
- **Commands:** `/vanish` - Toggle invisibility
- **Permissions:** `vanish.use`, `vanish.damage`
- **Download:** [umod.org/plugins/vanish](https://umod.org/plugins/vanish)

**Configuration:**
```json
{
  "Enable damage to vanished players": false,
  "Enable vanish on admin connect": true,
  "Show vanish status in chat": false
}
```

### Admin Radar
**Real-time player monitoring system**

- **Purpose:** Track all players on server map
- **Features:** Live player positions, ESP overlay, teleport to players
- **Commands:** `/radar` - Open admin radar interface
- **Perfect for:** Monitoring cheaters, helping lost players
- **Download:** [umod.org/plugins/admin-radar](https://umod.org/plugins/admin-radar)

## Player Features & Quality of Life

### Teleportation
**Complete teleport and home system**

- **Purpose:** Player teleportation and home management
- **Features:** Set homes, teleport to players, admin teleport
- **Commands:** `/home`, `/sethome`, `/tpr`, `/tpa`
- **Cooldowns:** Configurable delays prevent abuse
- **Download:** [umod.org/plugins/teleportation](https://umod.org/plugins/teleportation)

**Essential Settings:**
```json
{
  "Home - Max homes per player": 3,
  "TPR - Cooldown": 600,
  "Home - Cooldown": 300,
  "Block teleport in combat": true
}
```

### Kits
**Starter and VIP item packages**

- **Purpose:** Provide item kits to players
- **Features:** Multiple kit types, cooldowns, permissions
- **Commands:** `/kit`, `/kit list`, `/kit starter`

- **Download:** [umod.org/plugins/kits](https://umod.org/plugins/kits)

**Popular Kit Examples:**

## Installation Best Practices

### Plugin Installation Order

1. **Install uMod framework** first
2. **Add admin tools** (Vanish, Admin Hammer)
3. **Install core features** (Teleportation, Kits)
4. **Add economy system** (Economics, Shop)
5. **Configure permissions** for all plugins

### Performance Considerations

- **Start with essentials** - Add plugins gradually
- **Monitor server performance** - Check CPU/RAM usage
- **Regular updates** - Keep plugins current
- **Remove unused plugins** - Clean up regularly

### Configuration Tips

- **Test on development server** first
- **Backup configs** before major changes
- **Document custom settings** for team reference
- **Review logs regularly** for errors

## Frequently Asked Questions

<AccordionGroup>

<Accordion title="How many plugins can I install safely?">
Most servers run 10-20 plugins without issues. Monitor performance and add gradually to find your server's limit.
</Accordion>

<Accordion title="Do plugins affect server wipes?">
Plugin data may persist through map wipes but is usually cleared during full wipes. Check individual plugin documentation.
</Accordion>

<Accordion title="Can players see what plugins are installed?">
Players can see plugins through commands they have access to. Some plugins have hidden modes for admin-only features.
</Accordion>

<Accordion title="How do I update plugins safely?">
Backup your server, update plugins one at a time, and test functionality before updating multiple plugins simultaneously.
</Accordion>
