## Self-hosting a V Rising dedicated server

You can also alternatively run a server on your own computer/PC with [SteamCMD](https://developer.valvesoftware.com/wiki/SteamCMD) or from Steam library.

### System requirements

A single V Rising server requires at least **6GB RAM**, **2 CPU cores**, and **20GB of storage**.

### Installing game server

**On Linux:**
`steamcmd force_install_dir ./V Rising_server/ +login anonymous app_update 1829350 quit`

**On Windows:**
`steamcmd.exe +force_install_dir c:\SteamCMD\V Rising_server\ +login anonymous +app_update 1829350 +quit`

### Starting game server

Run the server by launching `StartServer.bat` (Windows) or `./StartServer.sh` (Linux)
