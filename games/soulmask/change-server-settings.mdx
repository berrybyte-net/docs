To change the settings of your [Soulmask server](https://berrybyte.net/soulmask-server-hosting), you need to verify yourself as an admin and then adjust the settings in-game. Here's how:

1. Open the **Startup** tab on the [game control panel](https://game.berrybyte.net)
<Frame>
    <img src="/images/startup-lime.png" />
</Frame>
2. Set an admin password in the highlighted field
    <Frame>
      <img src="/images/soulmask-admin-pw.png" />
    </Frame>
3. To open the console, press the `~` key in-game. 
4. Type `gm key <yourPassword>`
5. The next screen will prompt for your admin password again, and press **Become the Admin**
6. Once all done, you can change the server settings as you like!
