## Self-hosting a Soulmask dedicated server

You can also alternatively run a server on your own computer/PC with [SteamCMD](https://developer.valvesoftware.com/wiki/SteamCMD) or from Steam library.

### System requirements

A single Soulmask server requires at least **16GB RAM**, **4 CPU cores**, and **20GB of storage**.

### Installing game server

**On Linux:**
`steamcmd force_install_dir ./Soulmask_server/ +login anonymous app_update 3017300 quit`

**On Windows:**
`steamcmd.exe +force_install_dir c:\SteamCMD\Soulmask_server\ +login anonymous +app_update 3017310 +quit`

### Starting game server

Run the server by launching `StartServer.bat` (Windows) or `./StartServer.sh` (Linux)
