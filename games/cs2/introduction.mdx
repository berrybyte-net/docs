## Frequently Asked Questions

<AccordionGroup>

<Accordion title="How do I connect to my CS2 server?">
Use the console command `connect [server-ip]:[port]` or add your server to favorites through the community server browser.
</Accordion>

<Accordion title="Can I install custom maps on my CS2 server?">
Yes, upload .bsp map files to your server's maps folder and change the map using `changelevel [mapname]` command.
</Accordion>

<Accordion title="How do I get admin access on my CS2 server?">
Use RCON with your server password or install an admin plugin like SourceMod for easier management.
</Accordion>

<Accordion title="What's the difference between CS2 and CS:GO servers?">
CS2 servers use updated Source 2 engine with improved graphics, physics, and networking compared to CS:GO's Source engine.
</Accordion>
