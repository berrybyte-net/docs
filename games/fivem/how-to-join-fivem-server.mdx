# How to Join a FiveM Server

## Getting Your Server IP

If you have a [FiveM server](https://berrybyte.net/fivem-server-hosting) with berrybyte:

1. Login to the [game control panel](https://game.berrybyte.net)
2. Copy the IP:port shown below your server name

## Connecting to Your Server

1. Launch FiveM
2. Press **F8** to open console
3. Type: `connect [your-server-ip]:[port]`
4. Press **Enter**

Example: `connect 192.168.1.100:30120`

## Alternative Method

1. Open FiveM
2. Click **Direct Connect**
3. Enter your server IP and port
4. Click **Connect**

## Common Issues

**Connection timed out**: Check your [FiveM server hosting](https://berrybyte.net/fivem-server-hosting) is online in control panel

**Server full**: Wait and try again later

**Stuck loading**: Wait 2-3 minutes for resources to download
