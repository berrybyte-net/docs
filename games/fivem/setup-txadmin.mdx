# How to Setup txAdmin on Your FiveM Server

## Installing txAdmin

1. Login to your [FiveM server hosting](https://berrybyte.net/fivem-server-hosting) control panel
2. Go to **Mods** tab
3. Find **txAdmin** and click **Install**
4. **Restart** your server

## Setting up txAdmin

1. Start your [FiveM server](https://berrybyte.net/fivem-server-hosting)
2. Look in the console for a message like: `Please open http://your-ip:40120 and complete the setup`
3. Copy the URL and open it in your web browser
4. Fill in the setup form:
   - Choose a username and password
   - Set your server name
   - Configure basic settings
5. Click **Save** to complete setup

## Accessing txAdmin

After setup, visit `http://your-server-ip:40120` and login with your credentials to manage your server.
