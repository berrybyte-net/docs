# How to Setup txAdmin on Your FiveM Server

## Installing txAdmin

1. Login to your [FiveM server hosting](https://berrybyte.net/fivem-server-hosting) control panel
2. Go to **Mods** tab
3. Find **txAdmin** and click **Install**
4. **Restart** your server

## Setup txAdmin

1. Check your [FiveM server](https://berrybyte.net/fivem-server-hosting) console for txAdmin URL (example: `http://your-ip:40120`)
2. Open the URL in your browser
3. Create admin username and password
4. Complete setup wizard

## Accessing txAdmin

Visit `http://your-server-ip:40120` and login with your credentials.

## What txAdmin Does

- Start/stop/restart server
- Kick/ban players
- Manage resources
- View server console
- Monitor performance

## Common Issues

**Can't access txAdmin**: Check port 40120 is open

**Login not working**: Reset password in server console
