Set a custom server icon to make your server stand out in the multiplayer server list.

## Server Icon Requirements

- **Size**: 64x64 pixels (use [imageresizer.com](https://imageresizer.com/) to resize)
- **Format**: PNG
- **Name**: `server-icon.png`

## Upload Your Icon

1. Save your icon as `server-icon.png` (64x64px)
2. Upload it to the main directory of your server
3. Restart your server
