## Self-hosting a Satisfactory dedicated server

You can also alternatively run a server on your own computer/PC with [SteamCMD](https://developer.valvesoftware.com/wiki/SteamCMD) or from Steam library.

### System requirements

A single Satisfactory server requires at least **8GB RAM**, **4 CPU cores**, and **15GB of storage**.

### Installing game server

**On Linux:**
`steamcmd +force_install_dir ./satisfactory_server/ +login anonymous +app_update 1690800 +quit`

**On Windows:**
`steamcmd.exe +force_install_dir c:\SteamCMD\satisfactory_server\ +login anonymous +app_update 1690800 +quit`

### Starting game server

Run the server by launching the FactoryServer executable with appropriate parameters.
