# Server-Fehlerbehebungsanleitung

Hast du Probleme mit deinem Game-Server? Diese umfassende Fehlerbehebungsanleitung deckt die häufigsten Probleme und ihre Lösungen ab. Egal, ob du mit Verbindungsproblemen, Performance-Problemen oder Server-Abstürzen zu tun hast, wir helfen dir, schnell wieder zum Laufen zu kommen.

## Schnelle Diagnose-Checkliste

Bevor du in spezifische Probleme eintauchst, gehe diese schnelle Checkliste durch:

- [ ] Zeigt dein Server als "Online" im Control Panel?
- [ ] Verwendest du die korrekte Server-IP-Adresse?
- [ ] Ist dein Spiel-Client auf die neueste Version aktualisiert?
- [ ] Hast du deinen Server kürzlich neu gestartet?
- [ ] Gibt es Fehlermeldungen in der Server-Konsole?

## Häufige Verbindungsprobleme

### Kann nicht zum Server verbinden

**Symptome**: Spiel-Client zeigt "Verbindung fehlgeschlagen", "Server nicht gefunden" oder Timeout-Fehler.

**Lösungen**:

1. **Server-Status verifizieren**
   - Überprüfe, ob Server in deinem Control Panel online ist
   - Schaue nach Fehlermeldungen in der Konsole
   - Starte den Server neu, falls er hängt

2. **Server-IP und Port überprüfen**
   - Kopiere die exakte IP-Adresse aus deinem Panel
   - Stelle sicher, dass du die korrekte Port-Nummer verwendest
   - Einige Spiele erfordern spezifische Port-Formate

3. **Firewall- und Netzwerkprobleme**
   - Deaktiviere vorübergehend deine Firewall zum Testen
   - Überprüfe, ob dein ISP Gaming-Ports blockiert
   - Versuche, dich von einem anderen Netzwerk zu verbinden

4. **Spiel-Versions-Konflikt**
   - Stelle sicher, dass dein Spiel-Client zur Server-Version passt
   - Aktualisiere deinen Spiel-Client falls nötig
   - Überprüfe Server-Version im Control Panel

### Server zeigt Offline, obwohl er Online sein sollte

**Symptome**: Control Panel zeigt Server als offline trotz Start.

**Lösungen**:

1. **Server-Neustart erzwingen**
   - Stoppe den Server vollständig
   - Warte 30 Sekunden
   - Starte den Server wieder

2. **Server-Ressourcen überprüfen**
   - Überwache RAM- und CPU-Nutzung
   - Upgrade falls konstant bei 100%
   - Lösche unnötige Dateien

3. **Server-Logs überprüfen**
   - Schaue in der Konsole nach Fehlermeldungen
   - Suche nach Absturzberichten
   - Identifiziere problematische Plugins/Mods

## Performance-Probleme

### Server-Lag und niedrige Performance

**Symptome**: Hohe Latenz, verzögerte Antworten, ruckeliges Gameplay.

**Lösungen**:

1. **Ressourcen-Optimierung**
   - Überwache RAM-Nutzung (sollte unter 80% bleiben)
   - Überprüfe CPU-Auslastung
   - Erwäge Plan-Upgrade

2. **Plugin/Mod-Verwaltung**
   - Deaktiviere unnötige Plugins/Mods
   - Aktualisiere alle Plugins auf neueste Versionen
   - Entferne konfliktverursachende Modifikationen

3. **Welt-Optimierung**
   - Reduziere Weltgröße falls möglich
   - Lösche ungenutzte Chunks
   - Begrenze Entity-Spawning

4. **Server-Einstellungen tunen**
   - Passe Sichtweiten-Einstellungen an
   - Optimiere Tick-Rate-Einstellungen
   - Konfiguriere Garbage Collection (Java-Spiele)

### Hoher Speicherverbrauch

**Symptome**: Server geht RAM aus, häufige Abstürze, langsame Performance.

**Lösungen**:

1. **Speicher-Zuteilung**
   - Überprüfe aktuelle RAM-Nutzung im Panel
   - Upgrade auf höheren RAM-Plan falls nötig
   - Optimiere Java-Heap-Einstellungen (Minecraft)

2. **Memory Leaks**
   - Starte Server regelmäßig neu
   - Aktualisiere Plugins/Mods
   - Entferne problematische Modifikationen

3. **Weltgrößen-Verwaltung**
   - Verwende World-Border-Plugins
   - Generiere Chunks effizient vor
   - Räume alte/ungenutzte Bereiche auf

## Server-Abstürze und Fehler

### Häufige Server-Abstürze

**Symptome**: Server stoppt unerwartet, erfordert manuellen Neustart.

**Lösungen**:

1. **Absturz-Ursachen identifizieren**
   - Überprüfe Absturz-Logs in der Konsole
   - Schaue nach Out-of-Memory-Fehlern
   - Suche nach Plugin/Mod-Konflikten

2. **Alles aktualisieren**
   - Aktualisiere Server-Software
   - Aktualisiere alle Plugins/Mods
   - Aktualisiere Java-Version (falls zutreffend)

3. **Hardware-Probleme**
   - Überwache Server-Ressourcen
   - Überprüfe auf Überhitzung
   - Kontaktiere Support bei Hardware-bezogenen Problemen

### Startup-Fehler

**Symptome**: Server startet nicht, zeigt Fehlermeldungen beim Booten.

**Lösungen**:

1. **Konfigurationsprobleme**
   - Überprüfe server.properties Datei
   - Verifiziere alle Dateiberechtigungen
   - Setze auf Standard-Konfiguration zurück falls nötig

2. **Datei-Korruption**
   - Stelle von aktuellem Backup wieder her
   - Lade Server-Dateien erneut hoch
   - Verifiziere Datei-Integrität

3. **Abhängigkeitsprobleme**
   - Installiere erforderliche Abhängigkeiten
   - Überprüfe Java-Versions-Kompatibilität
   - Aktualisiere Server-Software

## Erweiterte Fehlerbehebung

### Server-Konsole verwenden

Die Server-Konsole ist dein bester Freund für die Fehlerbehebung:

1. **Logs lesen**
   - Schaue nach ERROR- oder WARN-Nachrichten
   - Notiere Zeitstempel von Problemen
   - Suche nach spezifischen Fehlercodes

2. **Befehle ausführen**
   - Verwende Admin-Befehle zur Problemdiagnose
   - Überprüfe Spielerzahlen und Verbindungen
   - Überwache Ressourcenverbrauch

3. **Log-Analyse**
   - Lade Logs für detaillierte Analyse herunter
   - Verwende Log-Analyse-Tools
   - Teile Logs mit Support wenn nötig

### Netzwerk-Diagnose

**Konnektivität testen**:

1. **Ping-Test**
   ```bash
   ping deine-server-ip
   ```

2. **Port-Testing**
   - Verwende Online-Port-Checker
   - Teste von mehreren Standorten
   - Verifiziere Port-Weiterleitung

3. **Traceroute-Analyse**
   ```bash
   traceroute deine-server-ip
   ```

### Dateisystem-Probleme

**Häufige Datei-Probleme**:

1. **Berechtigungen**
   - Stelle richtige Dateiberechtigungen sicher
   - Überprüfe Besitzer-Einstellungen
   - Setze Berechtigungen zurück falls nötig

2. **Festplattenspeicher**
   - Überwache verfügbaren Speicher
   - Räume alte Logs und Backups auf
   - Upgrade Speicher falls nötig

3. **Datei-Korruption**
   - Verifiziere Datei-Integrität
   - Stelle von Backups wieder her
   - Lade korrupte Dateien erneut hoch

## Notfall-Verfahren

### Server-Down-Notfall

1. **Sofortige Aktionen**
   - Überprüfe Server-Status im Panel
   - Versuche Server-Neustart
   - Schaue nach Wartungsbenachrichtigungen

2. **Falls Neustart fehlschlägt**
   - Überprüfe Konsolen-Logs
   - Kontaktiere sofort Support
   - Informiere Spieler über das Problem

3. **Daten-Wiederherstellung**
   - Stelle von neuestem Backup wieder her falls nötig
   - Dokumentiere jeden Datenverlust
   - Implementiere Präventionsmaßnahmen

### Sicherheitsvorfälle

1. **Vermuteter Einbruch**
   - Ändere sofort alle Passwörter
   - Überprüfe Zugriffs-Logs
   - Kontaktiere Support für Untersuchung

2. **Bösartige Aktivität**
   - Banne problematische Spieler
   - Stelle von sauberem Backup wieder her
   - Implementiere zusätzliche Sicherheitsmaßnahmen

Denk daran: Die meisten Server-Probleme können schnell mit dem richtigen Ansatz gelöst werden. Zögere nicht, unser Support-Team zu kontaktieren, falls du Hilfe brauchst!
