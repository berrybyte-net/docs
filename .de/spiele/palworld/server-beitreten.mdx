# Wie man einem Palworld Server beitritt

Diese Anleitung zeigt dir, wie du dich mit deinem Palworld-Server für Multiplayer-Gameplay mit Freunden verbindest.

## Die IP-Adresse finden

Bevor du deinem [Palworld-Server](https://berrybyte.net/palworld) beitreten und dich verbinden kannst, musst du die [IP-Adresse](https://whatismyipaddress.com/ip-address) finden (eine Kombination aus Zahlen, z.B. `104.167.215.1`).

1. Logge dich in das [Game Control Panel](https://game.berrybyte.net) ein
2. Finde die Subdomain oder IP:Port unter dem Namen des Servers
<Frame>
  <img src="/images/general/join-server.png" />
</Frame>
3. Klicke auf den hervorgehobenen Bereich und die IP wird in deine Zwischenablage kopiert


## Dem Multiplayer Palworld-Server beitreten
1. Starte Palworld
2. Wähle **Join Multiplayer Game** aus dem Hauptmenü
<Frame>
  <img src="/images/games/palworld/join-palworld-server.png" />
</Frame>
3. Gib die IP-Adresse und den Port deines Servers in das Feld unten ein
5. Klicke auf **Connect**, was dich verbindet und in deinen Multiplayer-Server lädt!

Gib dem Server ein paar Momente Zeit, die Verbindung zu akzeptieren und dann voilà, du solltest jetzt deinem Palworld-Server beitreten können!

## Häufige Probleme

- **"Failed to connect to server" Fehler**: Server könnte offline sein oder IP-Adresse falsch
  - **Lösung**: Überprüfe, ob der Server im Control Panel läuft und kontrolliere die IP-Adresse doppelt

- **Connection timeout beim Beitreten**: Netzwerkverbindung oder Firewall blockiert die Verbindung
  - **Lösung**: Überprüfe die Firewall-Einstellungen und stelle sicher, dass die Ports 8211 und 27015 zugänglich sind

- **Server erscheint nicht in der Liste**: Direkte Verbindung für Dedicated Server erforderlich
  - **Lösung**: Verwende die IP-Adressen-Verbindungsmethode anstatt des Server-Browsers

- **Version mismatch Fehler**: Client und Server laufen mit verschiedenen Palworld-Versionen
  - **Lösung**: Stelle sicher, dass sowohl Client als auch Server auf dieselbe Version aktualisiert sind
