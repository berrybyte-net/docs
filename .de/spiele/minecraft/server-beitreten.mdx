# Minecraft Server beitreten - Java und Bedrock Edition

Die Verbindung zu deinem Minecraft-Server ist der erste Schritt, um Multiplayer-Gameplay mit Freunden zu genießen. Diese Anleitung deckt alles ab, was du über das Beitreten zu Servern auf Java und Bedrock Editionen wissen musst - vom Finden deiner Server-IP-Adresse bis zur Fehlerbehebung häufiger Verbindungsprobleme.

## Die IP-Adresse deines Minecraft-Servers finden

Bevor du deinem [Minecraft-Server](https://berrybyte.net/minecraft-server-hosting) beitreten und dich verbinden kannst, musst du die [IP](https://whatismyipaddress.com/ip-address) (eine Kombination aus Zahlen, z.B. `104.167.215.1`) oder eine Subdomain (Buchstaben, z.B. `test.berrybyte.net`) finden.

1. Gehe zum [berrybyte Game Panel](https://game.berrybyte.net)
2. Finde die Subdomain oder IP:Port unter dem Namen des Servers
<Frame>
  <img src="/images/general/join-server.png" alt="berrybyte Game Panel mit Server-IP-Adresse hervorgehoben" />
</Frame>
3. Klicke auf den hervorgehobenen Bereich und die IP wird in deine Zwischenablage kopiert

## Einem Server auf Java Edition beitreten
1. Starte Minecraft
2. Wähle **Mehrspieler** aus dem Hauptmenü
3. Klicke **Server hinzufügen**, um den Server zu deiner Serverliste hinzuzufügen
4. Gib einen Namen und die IP:Port deines Servers in die Felder **Servername** und **Serveradresse** ein
5. Klicke auf den Server aus der Serverliste, um deinem Server beizutreten!

Alternativ kannst du **Direkte Verbindung** wählen und die Subdomain oder IP:Port in das Feld **Serveradresse** eingeben. Dies ermöglicht dir eine direkte Verbindung, ohne zuerst einen Server zur Serverliste hinzufügen zu müssen.

## Einem Server auf Bedrock Edition beitreten
1. Starte Minecraft und wähle **Spielen** aus dem Menü
2. Wähle den **Server**-Tab oben auf dem Bildschirm
3. Scrolle an den vorgestellten Servern vorbei und klicke **Server hinzufügen**
4. Fülle die Felder **Servername**, **Serveradresse** und **Port** aus und **Speichern**
5. Wähle deinen Server aus der Liste und klicke **Server beitreten**, um dem Server beizutreten!

Gib dem Server einen Moment Zeit, die Verbindung zu akzeptieren, und voilà - du solltest jetzt deinem Bedrock-Server beitreten können!

## Häufige Probleme

- **"Connection timed out" Fehler**: Server könnte offline sein oder Firewall blockiert Verbindung
  - **Lösung**: Überprüfe den Server-Status im Control Panel und stelle sicher, dass die IP-Adresse korrekt ist

- **"Can't resolve hostname" Fehler**: DNS-Probleme oder falsche Serveradresse
  - **Lösung**: Versuche die numerische IP-Adresse anstelle der Subdomain zu verwenden

- **"Outdated client/server" Nachricht**: Versionskonflikt zwischen Client und Server
  - **Lösung**: Stelle sicher, dass dein Minecraft-Client mit der Server-Version übereinstimmt

- **Bedrock-Spieler können Java-Server nicht beitreten**: Verschiedene Editionen sind standardmäßig nicht kompatibel
  - **Lösung**: Installiere das [Geyser Plugin](/games/minecraft/plugins/geyser), um plattformübergreifendes Spielen zu ermöglichen
