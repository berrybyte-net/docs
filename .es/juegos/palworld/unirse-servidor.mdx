# Cómo Unirse a un Servidor de Palworld

Unirse a un servidor de Palworld te permite explorar el mundo, capturar Pals y construir junto con amigos y otros jugadores. Esta guía te mostrará exactamente cómo conectarte a tu servidor de berrybyte y invitar a otros a la aventura.

## Encontrar la Información de tu Servidor

Antes de conectarte, necesitas obtener la información de conexión de tu servidor:

1. **Inicia sesión en tu Panel de Control de berrybyte**
2. **Selecciona tu servidor Palworld** de la lista
3. **Encuentra la información de conexión**:
   - **IP del servidor**: Ejemplo `play.berrybyte.net`
   - **Puerto**: Generalmente `8211` para Palworld
   - **Contraseña**: Si has configurado una

<Info>
La información completa del servidor se muestra en el panel de control. Asegúrate de que tu servidor esté en estado "En línea" antes de intentar conectarte.
</Info>

## Conectarse desde Steam (PC)

### Método 1: Conexión Directa (Recomendado)

1. **Abre Palworld desde Steam**
2. **Haz clic en "Unirse a Servidor Multijugador"**
3. **Selecciona "Conectar a Servidor Dedicado"**
4. **Ingresa la información del servidor**:
   - **Dirección**: `play.berrybyte.net:8211` (reemplaza con tu IP y puerto)
   - **Contraseña**: Ingresa si has configurado una
5. **Haz clic en "Conectar"**

### Método 2: A través de la Lista de Servidores

1. **Abre Palworld**
2. **Ve a "Unirse a Servidor Multijugador"**
3. **Busca en la lista de servidores** por el nombre de tu servidor
4. **Filtra por región** si es necesario para encontrarlo más fácil
5. **Haz doble clic** en tu servidor para unirte

<Warning>
Si tu servidor no aparece en la lista pública, asegúrate de que esté configurado como servidor público en el panel de control, o usa el método de conexión directa.
</Warning>

## Conectarse desde Xbox Game Pass

### Para Servidores Compatibles con Xbox

1. **Abre Palworld en Xbox o PC (Game Pass)**
2. **Selecciona "Multijugador"**
3. **Elige "Unirse a Servidor Dedicado"**
4. **Ingresa la información del servidor**:
   - **IP del servidor**: Tu IP de berrybyte
   - **Puerto**: Generalmente `8211`
   - **Contraseña**: Si aplica

<Info>
Para que jugadores de Xbox puedan unirse, tu servidor debe estar configurado específicamente para compatibilidad con Xbox. Consulta nuestra [guía de configuración de Xbox](/games/palworld/setup-xbox-dedicated-server).
</Info>

## Configurar Contraseña del Servidor

Para mantener tu servidor privado:

1. **Ve al panel de control de tu servidor**
2. **Navega a "Configuración del Servidor"**
3. **Busca la opción "ServerPassword"**
4. **Establece tu contraseña deseada**
5. **Guarda los cambios y reinicia el servidor**

Los jugadores necesitarán esta contraseña para unirse.

## Solución de Problemas de Conexión

### "No se puede conectar al servidor"

#### Verificaciones Básicas:
- **Confirma que el servidor esté en línea** en el panel de control
- **Verifica la IP y puerto** - deben ser exactos
- **Revisa la contraseña** si has configurado una
- **Asegúrate de tener la versión correcta** de Palworld

#### Soluciones:
1. **Reinicia el servidor** desde el panel de control
2. **Verifica tu conexión a internet**
3. **Desactiva temporalmente el firewall/antivirus**
4. **Intenta conectarte desde otra red** para descartar problemas de ISP

### "Servidor lleno"
- Tu servidor ha alcanzado el límite máximo de jugadores
- Espera a que alguien se desconecte
- Considera actualizar a un plan con más slots de jugadores

### "Versión incompatible"
- **Actualiza tu cliente** de Palworld a la última versión
- **Verifica la versión del servidor** en el panel de control
- **Actualiza el servidor** si es necesario

### "Contraseña incorrecta"
- **Verifica la contraseña** con el administrador del servidor
- **Asegúrate de no tener espacios extra** al ingresarla
- **Distingue mayúsculas y minúsculas** si aplica

## Gestión de Jugadores (Para Administradores)

### Comandos Básicos de Admin

Para usar comandos de admin, primero debes [convertirte en administrador](/games/palworld/become-an-admin).

#### Gestionar Jugadores:
```
/KickPlayer [nombre_jugador]     # Expulsar jugador
/BanPlayer [nombre_jugador]      # Banear jugador
/UnBanPlayer [nombre_jugador]    # Desbanear jugador
```

#### Información del Servidor:
```
/ShowPlayers                     # Mostrar jugadores conectados
/Info                           # Información del servidor
/Save                           # Guardar el mundo manualmente
```

### Configurar Límites de Jugadores

1. **Ve a la configuración del servidor**
2. **Busca "MaxPlayers"**
3. **Establece el número deseado** (máximo según tu plan)
4. **Reinicia el servidor** para aplicar cambios
